# 포트폴리오 사이트 만들기 프로젝트 <br/>

하나의 웹사이트를 여러 가지 프레임워크를 사용하여 구축하였습니다<br/>

 
1. [react.js]를 이용하여 사이트를 제작합니다. <br/>
2. [vue.js]를 이용하여 사이트를 제작합니다.<br/>
3. [next.js]를 이용하여 사이트를 제작합니다.<br/>
 

## 사용 스택<br/>
- react(https://ko.legacy.reactjs.org/) 를 사용하여 사이트를 번들링하고 관리합니다.<br/>
- gsap(https://greensock.com/gsap) 를 이용하여 패럴랙스 효과를 줍니다.<br/>
- lenis(https://lenis.studiofreight.com/) 를 이용하여 스므스 효과를 구현합니다.<br/>
- netlify(https://www.netlify.com/) 를 통해 사이트를 배포합니다.<br/>
- git(https://github.com/) 을 사용하여 파일을 관리합니다.<br/>
- HTML, CSS 기반으로 웹사이트의 기본 레이아웃 설계하고, 웹 표준 및 웹 접근성을 준수하여 작업합니다. [ARIA(Accessible Rich Internet Applications)]<br/>

## 프로젝트 실행<br/>
- react를 설치합니다. `npx create-react-app 타이틀`<br/>
- react-router-dom을 설치합니다. `npm install react-router-dom`<br/>
- sass를 설치합니다. `npm install sass`<br/>
- gsap를 설치합니다. `npm install gsap`<br/>
- lenis를 설치합니다. `npm install @studio-freight/lenis`<br/>
